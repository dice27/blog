### rubyにおける main 関数の書き方

Rubyにおいて直接スクリプトファイルとして呼び出すときだけ実行される   
main 関数を記述する方法について。

#### __FILE__ 変数を使用すればよい。

```ruby
if __FILE__ == $0
　# main method
end
```

上記の様に記述すればモジュールとしてインポートされる時には実行されない。
直接スクリプトファイルとして呼び出すときだけ実行される main 関数の様な動作をする。
